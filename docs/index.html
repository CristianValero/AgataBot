Our Values

The values ​​that govern us when creating content

Open source

Everything belongs to everyone. Thanks to this philosophy, all of us can create a very robust service and continually improve it.

Simple to use

If it is simple, it is likely that many people use it. Thanks to that, the community can grow more easily.

Listening to requests

If you are not a developer, nothing happens. We also listen to your requests to incorporate them into the service and thus create a more flexible infrastructure.
